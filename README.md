# juego-2d

Comandos para ejecutar.

1.- Crear un usuario administrador

`python manage.py createsuperuser`

Seguir instrucciones en consola, no preocuparse si no ven que escribe, eso esta bien.

2.- Busco migraciones para la base de datos

`python manage.py makemigrations`

3.- Aplico las migraciones 

`python manage.py migrate`

4.- Levanto el proyecto

`python manage.py runserver`

5.- Ir al navegador y abrir la ruta

`http://localhost:8000`
